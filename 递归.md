####递归

***概论***
*函数直接或间接的调用自己*
*指定跳出条件,并携带结果跳出*
递归的思想,就是把一个未知问题转化为已知问题来不断逼近结果的思路;
将一个大任务,拆解为小任务,简单粗暴的解决的思路

**递归的技巧**
	* 假定递归函数已经写好
	* 寻找递推关系
	* 将递推关系转换为递归体
	* 将临界条件加入递归体

*示例*
求1-100的和(高斯加法)

假定已有函数sum(n)用于该问题;
```
var res = sum(100);
var	res = sum(99) + 100; 	
var res = sum(n-1) + n;			//抽象
```
将递归结构转换为递归体
```
function sum(n){
	return sum(n-1) + n;
}
```

将临界条件加入递归体
即,何时结束函数,返回结果--当n为1,返回结果
```
function sum(n){
	return (n<=1) ? 1 : sum(n-1) + n;
}
```
