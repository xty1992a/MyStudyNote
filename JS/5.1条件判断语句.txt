流程控制语句

	在JS中,程序默认从上至下一行行执行,
		我们可以通过流程控制语句改变程序执行顺序,使它能够根据不同的条件执行不同的语句,或者反复执行一段语句.
		-流程控制语句:
			1.条件判断语句,两个分支
			2.条件分支语句,多个分支
			3.循环语句
			
			
----------------------------
	-if语句,条件判断语句
		语法一:
			if (条件判断语句) {
								语句;
			}
		执行过程,首先执行if中的条件判断语句,
		得到true,则继续执行if后的语句.
		得到false,不继续执行.
		if语句只对紧随其后的语句或代码块起作用.使用{}将多条语句包括后可以使它们均对if生效.
		当if后的条件表达式的结果是一个非布尔值时,会将其转换为布尔值,然后在进行判断.
		
		语法二:
			if (条件判断语句) {
								语句;
			}else{
				语句
			}
		当if...else...执行时,首先执行if中的条件判断语句,
		得到true,则继续执行if后的语句.
		得到false,则执行else后的语句.
		
		语法三:
			if (条件判断语句) {
								语句;
			}else if{
				语句
			}else if{
				语句
			}else{
				语句
			}
		当if...else if...else...执行时,会自上向下依次对if中的条件表达式进行判断,如果满足任一个if的条件,则执行该if的语句,并结束判断.否则执行else后的语句.
		注意逻辑判断的顺序,谨防写出死代码.
